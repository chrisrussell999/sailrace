⚠️ Network Location Detected

Your device is using cell tower location instead of GPS satellites. This is not accurate enough for sailing navigation.

Fix for Pixel 8 / Android:

  1. Settings → Location → Location Services
  2. Turn OFF "Google Location Accuracy"
  3. Turn OFF "WiFi scanning" and "Bluetooth scanning"
  4. Go back → App permissions → Chrome/Browser → Location → "Allow only while using app"
  5. Go OUTDOORS with clear sky view
  6. Disable WiFi and Bluetooth completely
  7. Wait 30-60 seconds for satellite lock
  8. Use the "🛰️ Force GPS" button in navigation
